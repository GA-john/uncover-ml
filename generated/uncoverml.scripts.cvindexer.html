Create a cross validation fold index.

Usage: cvindexer [OPTIONS] POINTSPEC OUTFILE

  Create a cross validation fold index file from a pointspec. This outputs
  an HDF5 file with "Latitude", "Longitude", and "FoldIndices arrays. If the
  pointspec is a grid, then "FoldIndices" is a 2D array.

Options:
  --folds INTEGER  Number of folds for crossvalidation
  --quiet          Log verbose output
  --help           Show this message and exit.
