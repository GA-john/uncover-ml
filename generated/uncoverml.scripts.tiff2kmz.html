uncoverml.scripts.tiff2kmzÂ¶

Convert a Geotiff to a KMZ archive for display on Google and Cesium maps (e.g. National Map).

Usage: tiff2kmz [OPTIONS] TIFF

  Turn a geotiff into a KMZ that can be dragged onto an instance of Terria
  Map. This also constructs a JPEG of the Geotiff, as it is required for the
  KMZ.

Options:
  --outfile PATH      Output file name, if not specified input file name is
                      used
  --overlayname TEXT
  --help              Show this message and exit.
