uncoverml.scripts.gammasensorÂ¶

Run the Gamma sensor prediction/super-resolution pipeline.

Usage: gammasensor [OPTIONS] GEOTIFF

Options:
  --invert                        Apply inverse sensor model
  --apply                         Apply forward sensor model
  --height FLOAT                  height of sensor
  --absorption FLOAT              absorption coeff
  --impute                        Use the sensor model to impute missing
                                  values in the deconvolution
  --noise FLOAT                   noise coeff for the inverse transform.
                                  Increasing this will remove missing data
                                  artifacts at the cost of image sharpness
  -v, --verbosity [DEBUG|INFO|WARNING|ERROR]
                                  Level of logging
  -o, --outputdir TEXT            Location to output file
  --help                          Show this message and exit.
